# Campaign Tracker Dashboard

Интерактивный дашборд для отслеживания эффективности рекламных кампаний. Разработан как внутренний инструмент для ускорения ежедневной рутины медиабайера: вместо ручного свода в Excel — реактивные метрики и условные статусы в реальном времени.

## Зачем это нужно

Как медиабайеру/маркетологу, мне часто приходилось собирать данные по кампаниям из разных источников и сводить их вручную. Этот дашборд решает проблему:
- **Авторасчёт ROI, CPL, CTR** на лету при вводе данных.
- **Условная подсветка статусов**: красный — низкий ROI, жёлтый — пограничный, зелёный — норма.
- **Фильтрация по гео, вертикали, источнику** — чтобы быстро находить проблемные кампании.

## Что реализовано

- Реактивная панель статистики (Vue 3 + Composition API).
- Условные стили статусов на основе пороговых значений метрик.
- Компонентная архитектура (переиспользуемые карточки кампаний, графики-заглушки).
- Сборка на Vite — быстрая разработка и лёгкая деплой-стратегия.

## Стек технологий

- Vue 3 (Composition API)
- Vite
- HTML5 / CSS3
- JavaScript (ES6+)
